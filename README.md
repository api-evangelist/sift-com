# Sift

San Francisco-based Digital Trust and Safety platform (formerly Sift Science) protecting more than 700 global brands from payment fraud, account takeover, content abuse, promotion abuse, and money movement risk. Sift was founded in 2011, processes more than one trillion events per year, and reports a median annual loss prevented of about $4.2M per customer.

This API Evangelist profile catalogs Sift's developer-facing APIs, official SDKs, and the artifacts generated by the API Evangelist profiling pipeline (OpenAPI, JSON Schema, JSON-LD, Naftiko capabilities, Spectral rules, vocabulary, plans, rate limits, and FinOps).

## APIs

| API | Description | OpenAPI |
|---|---|---|
| Sift Events API | Primary integration surface. Submit reserved and custom events for scoring. | [openapi/sift-events-api-openapi.yml](openapi/sift-events-api-openapi.yml) |
| Sift Decisions API | Apply analyst or workflow decisions to users, orders, sessions, and content. | [openapi/sift-decisions-api-openapi.yml](openapi/sift-decisions-api-openapi.yml) |
| Sift Score API | Retrieve and recompute Sift Scores per abuse type. | [openapi/sift-score-api-openapi.yml](openapi/sift-score-api-openapi.yml) |
| Sift Verification API | Send, resend, and check OTP and biometric verification challenges. | [openapi/sift-verification-api-openapi.yml](openapi/sift-verification-api-openapi.yml) |
| Sift Workflows API | Inspect Sift Workflow runs and per-entity status. | [openapi/sift-workflows-api-openapi.yml](openapi/sift-workflows-api-openapi.yml) |
| Sift Labels API | Legacy fraud labels. Deprecated in favor of Decisions for new integrations. | [openapi/sift-labels-api-openapi.yml](openapi/sift-labels-api-openapi.yml) |
